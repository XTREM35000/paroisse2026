## 🎬 AJOUTER UNE VIDÉO (3 ÉTAPES)

```
1️⃣ Médias → Vidéos → [➕ Ajouter]

2️⃣ Remplir:
   📝 Titre
   📄 Description
   🎬 Fichier (MP4/WebM, max 500MB)
   🏷️ Catégorie

3️⃣ [Publier] ✅
```

---

## 📅 CRÉER UN ÉVÉNEMENT (4 ÉTAPES)

```
1️⃣ Admin → Événements → [➕ Créer]

2️⃣ Infos de base:
   📝 Titre
   📄 Description
   📍 Lieu

3️⃣ Dates:
   📅 Date & heure début
   ⏰ Heure fin

4️⃣ [Créer l'événement] ✅
```

---

## 👥 AJOUTER UN UTILISATEUR (3 ÉTAPES)

```
1️⃣ Admin → Utilisateurs → [➕ Ajouter]

2️⃣ Remplir:
   👤 Prénom, Nom
   📧 Email
   🔑 Mot de passe (min 8 caractères)
   🎭 Rôle (Membre/Modérateur/Admin)

3️⃣ ☑️ Envoyer invitation par email
   [Créer l'utilisateur] ✅
```

---

## ✏️ ÉDITER LA PAGE D'ACCUEIL

```
Admin → Paramètres généraux → [📝 Éditeur: Page d'accueil]

Vous pouvez modifier:
├─ 🖼️ Grande image (Hero)
├─ 📝 Titre et sous-titre
├─ 🎨 Sections du contenu
└─ [Enregistrer les modifications] ✅
```

---

## 💬 MODÉRER RAPIDEMENT

```
CHAT:
  Communauté → Chat
  Cliquez message → [⋮] → Supprimer/Avertir/Bannir

INTENTIONS DE PRIÈRE:
  Culte & Prière → Intentions
  Cliquez intention → ✅ Approuver ou ❌ Supprimer

COMMENTAIRES:
  Cliquez message → Actions: Approuver/Supprimer/Avertir
```

## 🆘 PROBLÈMES COURANTS

| Problème                 | Solution                           |
| ------------------------ | ---------------------------------- |
| Vidéo trop volumineuse   | Réduisez taille (max 500MB)        |
| Fichier refusé           | Utilisez MP4 ou WebM               |
| Pas de menu Admin        | Vérifiez que vous êtes Admin       |
| Mail non reçu            | Vérifiez SPAM, renvoyez invitation |
| Page ne s'enregistre pas | Vérifiez tous les champs requis    |
