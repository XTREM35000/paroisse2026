```
  ↓

  ├─ Ajouter un nouvel utilisateur
  ├─ Modifier le rôle d'un utilisateur
  ├─ Désactiver/Réactiver un compte
  └─ Réinitialiser un mot de passe
```

## 💡 CONSEILS DE LECTURE

### Pour un nouveau responsable

```
1. Lire: "Avant de commencer" du guide complet
2. Lire: Table des matières (orientation générale)
3. Parcourir: Quick Reference (actions courantes)
4. Lire par sections: Guide complet au besoin
5. Enregistrer: Bookmarks des pages les plus utilisées
```

---

### Pour une action spécifique

```
1. Chercher dans Quick Reference
   → Si trouvé: Utiliser la procédure simplifiée
   → Si non trouvé: Aller à l'étape 2

2. Chercher dans Table des matières
   → Cliquer sur le lien vers la section

3. Lire le guide complet pour cette section
   → Suivre les procédures détaillées
```

---

### Pour le dépannage

```
1. Chercher dans "Problèmes courants" (Quick Reference)
2. Si non trouvé → Chercher dans "Guide de dépannage" (Guide complet)
3. Si toujours pas résolu → Contacter support@faithflix.fr
```
