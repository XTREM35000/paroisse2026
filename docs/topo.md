# 📚 Guide d'Administration -Média Paroissiale

Bienvenue! Ce guide te permet d'administrer complètement le site **Faith Flix**. Ne t'inquiète pas, c'est simple et intuitif! 😊

---

## 🚀 Étape 1 : Se connecter

1. Accède au site : `https://paroisse-ten.vercel.app/` (ou l'adresse locale en développement)
2. Clique sur le bouton **"Connexion"** en haut à droite
3. Saisis tes identifiants :
   - **Email** : ton adresse email admin
   - **Mot de passe** : ton mot de passe sécurisé
4. Clique sur **"Se connecter"**

✅ **C'est fait!** Tu es connecté en tant qu'administrateur.

---

## 👁️ Étape 3 : Vue d'ensemble (Tableau de bord)

C'est ta **première page après connexion**. Elle te montre un résumé complet du site.

### Qu'est-ce que tu vois?

**En haut** : Une grande image de bannière

- 📸 Tu peux **changer cette image** en cliquant sur le bouton d'édition

**4 grandes cartes avec des chiffres** :

- 👥 **Membres** : Nombre total d'membres inscrits
- 🎥 **Vidéos** : Nombre de vidéos uploadées
- 📅 **Événements** : Nombre d'événements programmés
- 📊 **Total vues** : Nombre total de fois où les vidéos ont été regardées

**Un graphique en courbe** (à droite) :

- Montre l'**activité des 7 derniers jours**
- Les vues montent et descendent = c'est normal!

**4 vidéos récentes** (en bas à gauche) :

- Les vidéos les plus récemment uploadées
- Tu peux cliquer dessus pour les voir en détail

**Résumé rapide** (en bas à droite) :

- Un petit tableau récapitulatif des stats clés

### 💡 Conseil utile

Reviens ici régulièrement pour vérifier comment le site se porte!

## 📊 Étape 4 : Voir les Analytics (Statistiques détaillées)

Clique sur **"Analytics"** dans le sidebar pour accéder aux statistiques détaillées.

### Qu'est-ce qu'on voit?

**En haut** : Image de bannière (modifiable comme le tableau de bord)

**3 cartes importantes** :

1. 📈 **Vues cette semaine** : Total des vues sur 7 jours
2. 🚀 **Engagement** : Pourcentage de croissance par rapport à la semaine précédente
3. 🎥 **Vidéos totales** : Nombre total de vidéos

**Un graphique en courbe** (à gauche) :

- Montre les **vues quotidiennes** pendant 7 jours
- Plus haut = plus de gens regardent tes vidéos! 🎉

**Un graphique en camembert** (à droite) :

- Montre la **répartition par catégorie** :
  - Sermons
  - Musique
  - Enseignement
  - Célébration
  - etc.

**Top vidéos** (en bas à gauche) :

- Les **6 vidéos les plus regardées**
- Affiche : titre, vues, catégorie, date

**Alertes** (en bas à droite) :

- État du système (tout fonctionne? 🟢)
- Problèmes éventuels

## 🎥 Étape 5 : Gérer les Vidéos

Clique sur **"Vidéos"** dans la section **CONTENU** du sidebar.

### Qu'est-ce que tu peux faire?

#### ➕ Ajouter une vidéo

1. Clique sur le bouton **"+ Ajouter une vidéo"** (bleu, en haut)
2. Une fenêtre s'ouvre pour remplir :
   - 📝 **Titre** : Nom de la vidéo
   - 📄 **Description** : Texte explicatif
   - 🎬 **Fichier vidéo** : Sélectionne un fichier .mp4 ou .avi
   - 📸 **Miniature** : Image qui s'affiche avant de lancer la vidéo
   - 📂 **Catégorie** : Sermon, Musique, Enseignement, etc.
3. Clique **"Envoyer"** ✅

#### ✏️ Modifier une vidéo

1. Clique sur la vidéo que tu veux modifier
2. Change ce que tu veux (titre, description, catégorie...)
3. Clique **"Enregistrer"** ✅

#### ❌ Supprimer une vidéo

1. Clique sur le 🗑️ (poubelle) à côté de la vidéo
2. Confirme : "Êtes-vous sûr?" **→ Oui**
3. C'est supprimé! ✅

---

## 📸 Étape 6 : Gérer la Galerie (Photos)

Clique sur **"Galerie"** dans la section **CONTENU** du sidebar.

### Qu'est-ce que tu peux faire?

#### ➕ Ajouter une photo

1. Clique sur **"+ Ajouter une image"**
2. Remplis :
   - 📝 **Titre** : Nom de la photo
   - 📄 **Description** : Petite explication
   - 🖼️ **Fichier image** : Sélectionne une image .jpg, .png, etc.
   - 📂 **Catégorie** : (Événements, Baptême, Mariage, etc.)
3. Clique **"Envoyer"** ✅

#### 📖 Voir la galerie

- Tu vois toutes les photos en **petit format (vignettes)**
- Clique sur une photo pour la voir en grand

#### ✏️ Modifier une photo

- Clique sur une photo
- Modifie le titre ou la description
- Clique **"Enregistrer"** ✅

#### ❌ Supprimer une photo

- Clique sur le 🗑️ (poubelle)
- Confirme
- C'est supprimé! ✅

---

## 📅 Étape 7 : Gérer les Événements

Clique sur **"Événements"** dans la section **CONTENU** du sidebar.

### Qu'est-ce que tu peux faire?

#### ➕ Créer un événement

1. Clique sur **"+ Ajouter un événement"**
2. Remplis les informations :
   - 📝 **Titre** : Nom de l'événement (ex: "Messe du dimanche")
   - 📄 **Description** : Détails sur l'événement
   - 📍 **Lieu** : Où ça se passe (ex: "Église Saint-Paul")
   - 📅 **Date & Heure de début** : Quand ça commence
   - 🕒 **Heure de fin** (optionnel) : Quand ça finit
3. Clique **"Créer"** ✅

#### ✏️ Modifier un événement

1. Clique sur l'événement à modifier
2. Change ce que tu veux
3. Clique **"Enregistrer"** ✅

#### ❌ Supprimer un événement

1. Double-clic sur l'événement (confirmation de sécurité)
2. Clic à nouveau pour confirmer
3. C'est supprimé! ✅

## 👥 Étape 8 : Gérer l'Annuaire (Répertoire des membres)

Clique sur **"Annuaire"** dans la section **CONTENU** du sidebar.

### Qu'est-ce que tu vois?

Une **liste de tous les membres/services** de l'église :

- 👤 Nom
- 📞 Téléphone
- 📧 Email
- 🏢 Catégorie (Prêtre, Diacre, Chantre, etc.)
- ✉️ Site web (optionnel)

### Qu'est-ce que tu peux faire?

#### ➕ Ajouter un membre/service

1. Clique sur **"+ Ajouter"**
2. Remplis :
   - 👤 **Nom** : Nom complet
   - 📧 **Email** : Email de contact
   - 📞 **Téléphone** : Numéro de téléphone
   - 📝 **Description** : Qui c'est? Son rôle?
   - 📂 **Catégorie** : Prêtre, Musicien, Servant, etc.
   - 🌐 **Site web** : URL personnelle (optionnel)
   - 📸 **Photo** : Ajouter une photo (optionnel)
3. Clique **"Ajouter"** ✅

#### ✏️ Modifier un membre

1. Clique sur la fiche du membre
2. Change les informations
3. Clique **"Enregistrer"** ✅

#### ❌ Supprimer un membre

1. Clique sur le 🗑️ (poubelle) sur la fiche
2. Confirme
3. C'est supprimé! ✅

## 🏠 Étape 9 : Éditer la Page d'Accueil

Clique sur **"Accueil"** dans la section **ADMINISTRATION** du sidebar.

### Qu'est-ce que tu peux modifier?

#### 1️⃣ Bannière en haut

- 📸 Clique sur **"Changer l'image"** pour mettre une autre photo
- Sélectionne une image depuis ton ordinateur

#### 2️⃣ Titre et texte

- 📝 Modifie le **titre principal** de la page d'accueil
- ✏️ Modifie la **description/sous-titre**
- 💾 Clique **"Enregistrer"** ✅

#### 3️⃣ Sections de contenu

- Ajoute ou modifie des **sections avec titre et description**
- Chaque section peut avoir une image
- 💾 **Enregistre** après chaque modification

#### 4️⃣ Prévisualisation

- Clique sur **"Aperçu"** pour voir comment ça va ressembler
- Les visiteurs verront exactement ça! 👀

Clique sur **"À propos"** dans la section **ADMINISTRATION** du sidebar.

- 📸 Changer la **bannière**

## 👤 Étape 11 : Gérer les Membres (membres)

Clique sur **"Membres"** dans la section **ADMINISTRATION** du sidebar.

### Qu'est-ce que tu vois?

Une liste de tous les **membres inscrit** sur le site :

- 👤 Nom et email
- 🔐 Rôle (Admin, membre)
- 📅 Date d'inscription
- ✅ État du compte

### Qu'est-ce que tu peux faire?

#### 👀 Voir les détails d'un membre

1. Clique sur le nom d'un membre
2. Tu vois :
   - Profil complet
   - Commentaires qu'il a laissés
   - Activité sur le site
3. Clique sur **"← Retour"** pour revenir à la liste

#### 🔄 Changer le rôle

1. Clique sur un membre
2. Change son **rôle** : Admin ou membre
3. Clique **"Enregistrer"** ✅

#### ❌ Supprimer un membre

1. Clique sur le 🗑️ (poubelle)
2. Confirme : "Êtes-vous sûr?"
3. C'est supprimé! ✅

#### 🚫 Bloquer un membre (optionnel)

Si quelqu'un pose problème (spam, insultes...):

1. Va sur son profil
2. Clique sur **"Bloquer"**
3. Il ne peut plus poster de commentaires

## 💬 Étape 12 : Modérer les Commentaires

Clique sur **"Commentaires"** dans la section **ADMINISTRATION** du sidebar.

### Qu'est-ce que tu fais?

Tu vérifies que les **commentaires sont appropriés** :

✅ **Bon commentaire** → Laisse tel quel
❌ **Commentaire offensant, spam ou hors sujet** → Supprime-le

### Comment supprimer un commentaire?

1. Trouve le commentaire problématique
2. Clique sur le 🗑️ (poubelle) à côté
3. Confirme
4. C'est supprimé! ✅

### 💡 Conseil

C'est important pour garder une **communauté respectueuse et positive**!

## 🎨 Étape 14 : Changer le Thème

En bas à droite (ou en haut à droite selon l'écran), tu trouveras un **toggle thème** :

## ❓ J'ai une question / Ça ne marche pas!

### Avant de paniquer :

1. **Recharge la page** (F5 ou Ctrl+R)
2. **Vérife ta connexion internet** 📡
3. **Vide le cache** de ton navigateur
4. **Réessaie** 🔄

### Si c'est toujours bloqué :

- 📧 Envoie un email au **développeur** avec :
  - Description du problème
  - Quelle section tu utilisais
  - Quelle erreur tu vois (si applicable)
  - Capture d'écran (si possible)
