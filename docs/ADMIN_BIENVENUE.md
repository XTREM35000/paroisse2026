## ⏱️ COMMENCEZ ICI (5 MINUTES)

### Étape 1 : Vérifier votre accès

```
1. Connectez-vous au siteMédia Paroissiale
2. Regardez à gauche: Voyez-vous "Tableau de bord" et "Administration"?
   ✅ OUI  → Allez à l'étape 2
   ❌ NON  → Contactez le chef IT
```

### Étape 2 : Comprendre la structure

```
Le menu latéral (Sidebar) est organisé en 6 GROUPES:

📊 Groupe 1: Tableau de bord (stats, analytics)
🎬 Groupe 2: Médias (vidéos, photos, podcasts)
⛪ Groupe 3: Culte & Prière (messes, homélies)
👥 Groupe 4: Communauté (chat, événements, annonces)
💰 Groupe 5: Donations (dons, campagnes)
⚙️ Groupe 6: Administration (gestion du site)

→ Chaque groupe a son propre guide détaillé!
```

### Étape 3 : Essayer une action simple

```
Essayez MAINTENANT:
1. Allez à: Tableau de bord → Vue d'ensemble
2. Regardez les statistiques
3. Revenez au menu principal
4. Explorez d'autres groupes

✅ Prêt? Continuez avec le guide!
```

## 🎯 VOS RÔLES ET RESPONSABILITÉS

### Si vous êtes **Administrateur complet** ⚙️

```
✅ Vous pouvez:
   - Créer et éditer du contenu (vidéos, photos, événements)
   - Gérer les utilisateurs (créer, modifier rôles)
   - Modérer les commentaires et le chat
   - Éditer les pages du site
   - Accéder à TOUS les menus

Responsabilités:
   - Vérifier régulièrement les commentaires
   - Approuver les contenus des utilisateurs
   - Gérer les permissions des autres admins
   - Signaler les bugs/problèmes
```

---

### Si vous êtes **Modérateur** 🛡️

```
✅ Vous pouvez:
   - Modérer les commentaires
   - Répondre aux utilisateurs
   - Voir les statistiques

❌ Vous ne pouvez pas:
   - Créer de contenu
   - Gérer les utilisateurs
   - Accéder à l'administration
```

---

### Si vous êtes **Secrétaire paroissiale** 📝

```
✅ Vous pouvez:
   - Ajouter des vidéos
   - Créer des événements
   - Gérer l'annuaire
   - Créer des annonces

❌ Vous ne pouvez pas:
   - Gérer les utilisateurs
   - Accéder aux paramètres avancés
```
